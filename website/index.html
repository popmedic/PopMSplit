About

PopMSplit allows it's user to "split" a movie (avi, flv, m4v, mkv, mov, mp4, ogg) into multiple different segments. Along with splitting the movie up, it can also be used to extract segments of audio and/or video from a movie file, and to convert containers.

PopMSplit uses ffmpeg -vcodec copy option to split your movies as quickly as you can copy them with 0 loss to quality. NO MORE RE-ENCODING!!!

But, if you want, or need to re-encode, then go ahead and do so, PopMSplit can do that too.

PopMSplit is currently only supported on Mac OS X 10.7.x. If demand for a Ubuntu, or Windows build is large enough, I can do so. Contact me below.

Install

  1. Download Pop Movie Splitter for OS X.
  2. Open PopMSplit.dmg by double clicking on it in finder.
  3. Drag PopMSplit into your Applications directory.
  4. Double click on Applications/PopMSplit

Documentation

Select Source Button


This is the starting point. Click this button to select the source movie that PopMSplit will be splitting.

Play/Pause Button


This button will begin the playing of the movie in the viewer, or pause the movie in the viewer.

Stop Button


This button will stop the playing of the movie in the viewer, and bring the slider back to 0.

Fast Back Button


This button will subtract one from the playback rate, causing the viewer to rewind quickly. The more times you hit this button the quicker it will rewind. Hit the Play/Pause Button to reset playback.

Fast Forward Button


This button will add one to the playback rate, causing the viewer to play faster. The more times you hit this button the faster it will play. Hit the Play/Pause Button to reset playback.

Add Split Mark Button


This button will add a split mark at the current spot in the movie to the Split Marks. These are the marks that are used to split the movie.

Remove Split Mark Button


This button will remove the currently selected split mark from the Split Marks. These are the marks that are used to split the movie.

Split Marks


This selector is a list of all the split marks added. By clicking and selecting a split mark the seek slider will jump to its location in the movie. Once selected you can remove the split mark by clicking the Remove Split Mark Button. This list of split marks is used as the marks to split the movie.

Split Button


This button will bring up the Spitting Window.

Splitting Window

Segments


This view lists all of your segments that will be created. You can double click on a filename to edit its name before creation. You can also select a segment and delete it using the Delete Segment Button.

Remove Segment Button


This button will remove the selected segment from the Segments view.

Audio/Video Checkboxes


These checkboxes control what streams you want split out. If you want a straight copy, leave both selected. If you want to pull out an audio sample as an mp3, then remove the check from the box next to video and leave the check in the audio box. If you want just the video without the audio remove the check mark from the audio box and leave the check mark in the video box.

Convert To


Here is where you can select to convert the segments to another encoding. This will drastically slow down the process, but sometimes you need an avi and all you have is a mp4. Place a check mark in the boxe next to Conver To and then select the file type to convert (re-encode) the segments too.
*Hint: You can use PopMSplit as a re-encoder! Just select the file you want to convert with the Select Source Button, don't set any Split Marks, just click on the Split Button, and check the Convert To box, and select the new encoding. Now when you go to split, it will just copy the source into the new encoding.

Start Splitting Button


This button will start splitting your segments up. When clicked it will change into a Stop Splitting Buttom and the Close Splitting Window Button will become disabled.

Stop Splitting Button


This button will stop the splitting up of your segments. When clicked it will cancel the current splitting, but previous segments will remain.

Exit Splitting Window Button


This button will close the Splitting Window.
